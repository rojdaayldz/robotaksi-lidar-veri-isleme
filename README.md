# 🚀 Unitree 4D LiDAR Tabanlı Haritalama ve Algılama Sistemi

Bu proje, Unitree 4D LiDAR sensörü kullanılarak gerçek zamanlı çevre algılama, veri toplama, haritalama ve nesne tespiti işlemlerini gerçekleştirmek amacıyla geliştirilmiştir.  

Proje kapsamında LiDAR verileri işlenerek ortamın 3D modellenmesi, mesafe ölçümü ve otonom sistemler için uygun veri altyapısının oluşturulması hedeflenmektedir.

---

# 📌 Proje Amacı

Bu projenin temel amacı:

- Unitree 4D LiDAR sensöründen veri almak
- Nokta bulutu (Point Cloud) verilerini işlemek
- Gerçek zamanlı çevre haritalaması yapmak
- Engel/Nesne tespiti gerçekleştirmek
- Otonom sistemlerde kullanılabilecek veri altyapısı geliştirmek

---

# 🛠 Kullanılan Teknolojiler

- Python
- ROS / ROS2
- Unitree 4D LiDAR SDK
- Open3D
- NumPy
- Matplotlib
- MQTT (Gerçek zamanlı veri iletişimi için)
